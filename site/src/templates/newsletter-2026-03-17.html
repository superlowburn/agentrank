What the data is telling you

Unity is dominating the top 5. Three of the top 15 are Unity MCP integrations. Game developers are adopting MCP faster than any other category right now, and the maintainers are keeping up. If you're building Unity-adjacent tooling, the ecosystem is further along than you'd guess.
The Go SDK is quietly essential. mcp-go (#6) has 8,400 stars but what pushed it there is dependents — a large number of other repos build on it. Stars follow attention; dependents follow actual usage. mcp-go has both.
fastmcp is the Python SDK that won. 24,000 stars and one of the freshest commit histories in the top 10. If you're writing Python MCP tools, this is the framework the ecosystem chose.
playwright-mcp is the most-installed tool in the ecosystem. 29,000 stars, actively maintained by Microsoft. It scores on both dimensions — star count and real-world adoption signals.

Ecosystem snapshot

Language distribution across all 25,632 indexed tools:

Python leads the index by volume, but TypeScript leads in the top 100. The tools that rise to the top tend to be TypeScript — it seems to correlate with more active maintenance cycles.
