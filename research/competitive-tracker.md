### Glama.ai

- **Tool count:** 19,503 MCP servers (largest directory surveyed)
- **Connectors registry:** 1,166 connectors (separate curated subset)
- **Scoring system:** A-F grades across Security, License, Quality — most developed quality-signal system of any directory
- **New feature:** "Sort by usage in the last 30 days" recently added
- **Other sorting:** Search Relevance, Recent Usage (30d), Date Added/Updated, Weekly Downloads, GitHub Stars
- **Filtering:** Environment (Remote: 8,486 / Local: 4,250 / Hybrid: 4,805), Language (Python: 8,218 / TypeScript: 6,718 / JS: 3,436), 50+ categories
- **Author status:** Official (934), Claimed (3,192)
- **Connector health tracking:** 447 Healthy / 724 Unhealthy / 315 Untested — only directory tracking server health
- **Per-call pricing:** Some connectors show per-call pricing ($0.01/call range), 10 free credits offer, x402/crypto payment integration
- **Pricing:** Free directory access; connector-level monetization layer exists
- **Traffic:** N/A
- **Urgent flag:** None — but A-F scoring system is their key moat and most directly comparable to AgentRank's composite score concept

### AgentRank Positioning Notes (2026-03-17)

- **Gap still open:** No competitor publishes a single composite score (0-100). Glama's A-F is per-dimension, not a unified rank. AgentRank's composite score with published methodology is a real differentiator.
- **AgentRank advantage on scale:** AgentRank reports 25K+ tools vs Glama's 19,503 (next largest). This is a credible lead worth publicizing.
- **Watch:** Glama's per-call connector pricing + health tracking = earliest signs of monetization in the space. Smithery's "Skills" product is strategic overlap with AgentRank v2.
- **DXT format:** MCP.so's early bet on Anthropic Desktop Extensions is worth tracking — if DXT takes off, directories with DXT coverage gain distribution.
