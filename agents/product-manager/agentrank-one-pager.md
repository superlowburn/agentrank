## The Problem

The MCP and agent tool ecosystem has grown to 25,000+ tools — and nobody can evaluate them. There is no G2, no Capterra, no npm star equivalent that reflects real quality. Developers pick tools by googling, guessing, and asking on Discord. Bad tools get installed. Good tools go undiscovered. Maintainers have no signal on how they rank against alternatives.

## The Solution

AgentRank is a daily-updated, scored index of every MCP server, agent tool, and AI skill across the ecosystem. Each tool gets an AgentRank score (0–100) built from five signals: GitHub stars, freshness, issue health, contributor count, and inbound dependents.

- **25,000+ tools indexed** across GitHub, npm, and the MCP ecosystem
- **13,000+ skills ranked** from skills.sh and cross-platform AI platforms
- **Daily pipeline** — scores refresh every 24 hours; new tools appear automatically
- **13 API endpoints** for programmatic access to rankings and tool metadata
- **MCP server on npm** — AI agents can query AgentRank directly during tool selection
- **Skill on skills.sh** — available across Claude Code, Cursor, Copilot, Cline, VS Code, Windsurf, and 15+ other platforms
- **41 SEO-optimized blog posts** driving organic developer traffic

## Why Now

Every major AI lab has adopted MCP. Anthropic shipped it first. OpenAI followed. Google is next. The ecosystem doubles every quarter. Developers are drowning in tools with no way to separate signal from noise.

There is no authoritative index. There is no reputation layer. That gap is AgentRank.
