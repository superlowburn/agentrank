## The CPM Argument (Use in Sales Conversations)

When a prospect pushes back on price, compare to alternatives:

| Channel | CPM (cost per 1,000 developer impressions) |
|---------|-------------------------------------------|
| Google Ads (developer targeting) | $15–40 |
| Sponsored GitHub README | $500–2,000 flat |
| DevRel conference booth | $5,000–20,000 |
| AgentRank Sponsored Listing | **$0.83–$1.50** |

At $299/mo with a conservative 200 incremental views: $1.50/view.
At $499/mo with 600+ incremental views: $0.83/view.

These are **high-intent** views — developers actively evaluating tools, not scrolling a feed.
