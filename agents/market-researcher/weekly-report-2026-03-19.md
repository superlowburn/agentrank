## Notable Trends & Patterns

### 1. Game Development as Emerging MCP Vertical
Four Unity MCP tools entered the index in one week — two from different developers tackling the same problem. This is a strong signal that Unity/game dev is becoming a first-class MCP use case. The duplication (CoplayDev vs CoderGamester) suggests competitive pressure and real developer demand.

### 2. Microsoft's MCP Investment Is Serious
Three Microsoft-owned tools in the top 15: `playwright-mcp`, `azure-devops-mcp`, and `mcp-for-beginners`. With playwright-mcp at 29,150 stars, Microsoft now holds the highest-star tool in the index. Their educational content (`mcp-for-beginners` at 15k stars) is driving ecosystem onboarding at scale.

### 3. MCP Framework Consolidation
FastMCP has two forks in the index — PrefectHQ/fastmcp (23,775 stars, the dominant version) and punkpeye/fastmcp (2,999 stars). The Prefect acquisition appears to have split the community. Score gap: 95.6 vs 89.7. PrefectHQ is winning on all signals.

### 4. Languages Expanding Beyond Python
`mark3labs/mcp-go` (Go) and `modelcontextprotocol/ruby-sdk` (Ruby) entering the same week signals real polyglot expansion. Go is showing strong early traction at 8,353 stars. Ruby SDK is at 751 stars but officially maintained by the MCP team — watch this one.

### 5. Infrastructure Layer Maturing
Tools like `samanhappy/mcphub` (MCP server management) and the earlier-indexed `inspector` tools suggest the ecosystem is building the supporting infrastructure layer, not just point tools. This is a maturity marker.
