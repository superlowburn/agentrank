# SEO Keyword Research: Unclaimed MCP Server Search Queries

**Date:** 2026-03-19
**Author:** Market Researcher (AUT-372)
**Context:** Competitor analysis (AUT-366) confirmed Glama has only ~2,500 organic monthly visits despite ~100K total visits. The "[tool name] MCP server" keyword space is largely unclaimed by any directory or ranking site.

---

## Executive Summary

The MCP server keyword landscape has split into two zones:

1. **Contested territory** — "best MCP server 2026", "top MCP servers", "how to build MCP server" — flooded by content farms (firecrawl.dev, builder.io, k2view.com, stackgen.com). Difficult to rank without significant domain authority.

2. **Blue ocean** — "[tool name] MCP server" queries, "MCP server ranking/score/quality" queries, and "best MCP server for [specific use case]" queries — answered poorly or not at all. Smaller sites (claudemcp.com, ai-stack.dev, shyft.ai) are ranking here with thin content that AgentRank's data-rich tool pages can beat.

**The core opportunity:** AgentRank is the only site that can answer "who's winning?" with real data. No competitor publishes a multi-signal ranked index. The queries that ask this question — "MCP server ranking", "best rated MCP servers", "most active MCP servers" — have almost no strong results. We own those queries by default. Everything else compounds from there.

**Volume note:** All search volumes are estimates derived from SERP analysis, keyword modifier patterns, and competitor traffic data. No paid SEO tool data was available.

## Competitive Positioning Summary

The competitors who rank for "[tool name] MCP server" queries are **claudemcp.com, ai-stack.dev, shyft.ai, and pulsemcp.com** — all lower domain authority sites with thin content (one-paragraph descriptions, no live data). AgentRank's tool pages include live score data, freshness signals, issue health ratios, contributor counts, and embed codes. That's a defensible content moat competitors can't easily replicate without building the scoring infrastructure.

The queries around **rankings and scores** ("MCP server ranking", "best rated MCP servers") are functionally uncontested. MCPMarket publishes daily top-10 lists sorted by GitHub stars only. No competitor publishes a multi-signal composite ranked leaderboard. These queries have a natural answer — AgentRank — and all we need is for the leaderboard page to be indexed and target those keywords explicitly.
