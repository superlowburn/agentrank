## Fields

**Tool Name:** AgentRank

**Tagline (max ~100 chars):**
Ranked index of 25,000+ MCP servers and AI agent tools, scored daily by real GitHub signals.

**Description:**
AgentRank crawls GitHub nightly and ranks every MCP server and AI agent tool using five composite signals: stars, freshness (days since last commit), issue health (closed/open ratio), contributor count, and inbound dependents. Each tool gets a score from 0–100. The leaderboard updates daily.

The problem it solves: GitHub has thousands of MCP servers, most of them abandoned. AgentRank separates the actively maintained, widely-trusted tools from the noise.

**Features:**
- 25,000+ MCP servers and agent tools indexed
- Daily score updates from live GitHub data
- Search, filter, and compare tools
- Free public API (13 endpoints, OpenAPI spec)
- MCP server — query rankings from Claude, Cursor, Copilot, or any MCP client
- Maintainer claim flow
