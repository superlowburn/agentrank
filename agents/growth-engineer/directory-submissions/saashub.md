**Full Description:**
AgentRank indexes every MCP server and AI agent tool on GitHub and ranks them daily using five real signals: stars, freshness, issue health, contributor count, and inbound dependents. Each tool gets a composite score from 0–100.

The leaderboard updates nightly. Developers can search, filter, and compare tools side-by-side. There's a free public API with 13 endpoints and an MCP server so AI agents can query the rankings directly from Claude, Cursor, Copilot, or any MCP-compatible client.
