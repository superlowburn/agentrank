**Short Description (tagline):**
Ranked index of 25,000+ MCP servers and AI agent tools, updated daily.

**Full Description:**
AgentRank is a daily-updated leaderboard of 25,000+ MCP servers and AI agent tools. It crawls GitHub nightly, scoring each tool across five real signals: popularity (stars), maintenance freshness, issue responsiveness, contributor diversity, and inbound dependents. Each tool gets a composite AgentRank score from 0–100.

Unlike raw GitHub search, AgentRank weights by quality signals — so actively maintained tools with engaged maintainers rank higher than stale high-star repos. Updated nightly.

**Features:**
- 25,000+ tools indexed
- Daily score updates
- Search, filter, compare
- Free public API (13 endpoints)
- MCP server (query from Claude, Cursor, Copilot)
- Maintainer claim flow
