## Fields

**Tool Name:** AgentRank

**Website:** https://agentrank-ai.com

**GitHub:** https://github.com/superlowburn/agentrank

**Category / Layer:** Developer Tools > Open Source Tools > Software Discovery

**Short Description:**
Daily-updated ranked index of 25,000+ MCP servers and AI agent tools, scored by real GitHub signals.

**Full Description:**
AgentRank crawls GitHub nightly and ranks every MCP server and AI agent tool using five composite signals: stars, freshness, issue health, contributor count, and inbound dependents. Each tool gets a score from 0–100.

Built for developers choosing between MCP tools: instead of sorting GitHub repos by stars alone, AgentRank weights signals by maintenance quality — a 10,000-star repo with no commits in 6 months scores lower than a 500-star repo with active weekly development.

Free public API (13 endpoints). MCP server available for AI agent usage directly from Claude, Cursor, or any MCP client.
