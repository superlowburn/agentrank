**Application Name:** AgentRank

**Website:** https://agentrank-ai.com

**Description (will appear on the listing page):**
AgentRank is a daily-updated ranked index of 25,000+ MCP servers and AI agent tools. It crawls GitHub nightly and scores every tool across five signals: popularity (stars), freshness (days since last commit), issue health (closed/open ratio), contributor count, and inbound dependents. Each tool gets a composite AgentRank score from 0–100.

The goal is signal over noise. GitHub has thousands of MCP servers — most are abandoned. AgentRank separates actively maintained, widely-trusted tools from the rest. Developers can search, filter by category, compare tools, and access rankings via a free public API or MCP server.
